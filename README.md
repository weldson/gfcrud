### About the project

This project has the porpose of manage employees from a company. A user can manage employees and companies, and enroll a employe in a existing company.
### Getting Started

To run the current project is necessary to run the following commands:

```typescript
    npm install

    npx knex migrate:latest

    npx knex seed:up
```
